# 📊 Bash Script DataBase Management System

---

## 🌟 Overview
This project is a **Bash-based database management tool** that lets you manage simple text-based tables entirely from the command line.  
It’s designed for learning and small-scale data organization, offering a lightweight alternative to full database systems.  

With this script, you can **add, update, delete, insert, and view records** in different tables with ease. Perfect for school projects, practice, or simple data management! 🎯

---

## 🚀 Features
- **➕ Add Records**: Insert new entries into any table quickly.  
- **✏️ Update Records**: Modify existing records safely.  
- **🗑️ Delete Records**: Remove unwanted entries without affecting other data.  
- **📄 View Data**: Display all records in a table, with optional exclusion of the header line for clean output.  
- **🔍 Search Records**: Find records by column values or keywords easily.  
- **📊 Sort Records**: Organize table data by specific columns.  
- **📚 Multiple Table Support**: Works with `students`, `courses`, `departments`, `teachers`, and any new tables you add.  
- **⚡ Lightweight**: Runs directly in Bash, no external dependencies.  
- **🔧 Extendable**: Easily add new tables or expand features without breaking existing functionality.  
- **💾 Auto Backup (Optional)**: Save table backups before modifying data.  
- **🧹 Clean Data**: Automatic trimming of extra spaces and validation of input.  
- **👀 Interactive CLI**: User-friendly prompts guide you through every action.  
- **🛡️ Safe Operations**: Validates input to prevent accidental data corruption.

---

## ⚙️ Requirements
- A system with **Bash** support (Linux, macOS, or Windows with WSL).  
- Basic knowledge of the **command line**.  
- Read and write permissions in the project folder. ✅

---

## 📝 How to Use
1. Open a terminal in the project directory.  
2. Make the script executable (if necessary):
```bash
chmod +x DBMS.sh
```
3. Run the script:
```bash
./DBMS.sh
```
4. Follow the interactive prompts to add, update, delete, or view records. 🖱️
